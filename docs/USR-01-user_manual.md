# [USR-01] 操作マニュアル (User Operations Manual) - ゆうぞら (Yuzora)

本マニュアルは、青空文庫縦書きビューアー「ゆうぞら」の基本的な使い方、画面表示の見方、および表示設定のカスタマイズ方法について説明します。

---

## 1. 読書の開始方法 (Getting Started)

ゆうぞらを起動すると、最初に**ウェルカム画面**が表示されます。以下のいずれかの方法で作品を読み込むことで、読書を開始できます。

### 1.1 プリロード（事前定義）作品から選択する
- 画面中央の「作品を選択して読む」エリアに、吉川英治「宮本武蔵」全8巻のカード形式のリストが表示されています。
- 読みたい巻のカードをタップ（またはクリック）するだけで、即座に縦書きビューアー画面へ遷移し、読書を開始できます。

### 1.2 自分のファイルを読み込む
お手持ちの青空文庫形式のテキストファイル（`.txt`）やHTML/XHTMLファイル（`.html`, `.xhtml`）を読み込んで表示することができます。
- **ドラッグ＆ドロップ**: パソコン環境では、ブラウザのウインドウ（点線のドラッグエリア）内に直接ファイルをドロップするだけでロードできます。
- **ファイル選択ダイアログ**: 「ファイルを選択」ボタンを押し、ファイル選択画面からファイルを選んで読み込みます。
> [!NOTE]
> 青空文庫標準の文字コードである Shift_JIS (CP932) 形式および一般的な UTF-8 形式のいずれのファイルであっても、自動的に文字コードを判定して文字化けすることなく読み込まれます。

## 2. 読書画面の操作方法 (Reader Navigation)

読書画面では、書籍のように「1ページずつ」めくって読み進めることができます。

### 2.1 ページをめくる（進む・戻る）

#### A. 画面の端をクリック/タップする
- **右から左（RTL/標準）送り時**:
  - 画面の**左エリア**をクリック/タップすると、**次のページ（左方向）**に進みます。
  - 画面の**右エリア**をクリック/タップすると、**前のページ（右方向）**に戻ります。
- **左から右（LTR）送り時**:
  - 画面の**右エリア**をクリック/タップすると、**次のページ（右方向）**に進みます。
  - 画面の**左エリア**をクリック/タップすると、**前のページ（左方向）**に戻ります。

#### B. キーボード操作 (PC)
- **RTL（標準）**: `←` 矢印キーで次のページに進み、`→` 矢印キーで前のページに戻ります。
- **LTR**: `→` 矢印キーで次のページに進み、`←` 矢印キーで前のページに戻ります。
- **メニュー切替**: `↑`（上）または `↓`（下）矢印キーで、画面上下のメニュー（ヘッダー・フッター）の表示・非表示を切り替えることができます。

#### C. タッチスワイプ操作 (スマートフォン・タブレット)
- タッチデバイス上で指を左右に滑らせることで、滑りすぎることなく正確に「1ページ分」めくられます。
- **RTL（標準）**: **右スワイプ**（指を左から右へ動かす）で**次のページ**に進み、**左スワイプ**（指を右から左へ動かす）で**前のページ**に戻ります。
- **LTR**: **左スワイプ**（指を右から左へ動かす）で**次のページ**に進み、**右スワイプ**（指を左から右へ動かす）で**前のページ**に戻ります。

## 3. ヘッダー・フッターメニューの利用 (Header & Footer UI)

本文表示領域の中央付近をタップ（またはクリック）すると、上部に**ヘッダーメニュー**、下部に**フッターメニュー**がスライド表示されます。メニューが表示された状態で再度本文をタップするか、操作を停止して3秒経過すると自動的に非表示になり、読書に集中できます。

### 3.1 ヘッダーメニューの機能
- **「戻る」ボタン (`<`)**: 読書画面を終了し、ウェルカム画面に戻ります。
- **作品タイトル**: 現在読んでいる作品名が表示されます。
- **「最初に戻る」ボタン (`>>` または `<<`)**: 
  - 瞬時に作品の1ページ目（先頭）へ移動します。
  - ページの送り方向に応じてボタンの向きが変化します（RTL時は右端の開始位置を指す `>>`、LTR時は左端の開始位置を指す `<<`）。
- **「設定」ボタン (`⚙`)**: 画面右側から表示カスタマイズ設定パネル（ドロワー）を引き出します。

### 3.2 フッターメニューと進捗操作の機能
- **ページ数表示 (例: `5 / 120 ページ`)**:
  - 現在のページ番号と総ページ数が表示されます。
  - **この文字エリアをクリック/タップすると、ページ指定ダイアログが表示され、任意のページ番号を直接入力してジャンプできます。**
- **プログレスバーと進捗率 (例: `4%`)**:
  - 本全体の読了進捗状況を視覚的に示します。
  - **ページの送り方向連動**: RTL設定時は**右端（0%）から左端（100%）へ向けて**進捗が増加します。LTR設定時は**左端（0%）から右端（100%）へ向けて**進捗が増加します。
  - **タップジャンプ**: プログレスバー上の任意の位置をクリック/タップすると、その割合のページへ直接ジャンプします。
  - **ドラッグ（スクラブ）ジャンプ**: バー上の丸い**つまみ（目印）**を指やマウスでドラッグして左右に動かすことで、本をパラパラとめくるようにリアルタイムで読書位置を切り替えることができます。

## 4. 表示のカスタマイズ設定 (Settings Customization)

ヘッダーの `⚙` アイコンを押すと表示される**表示設定ドロワー**から、以下の設定をリアルタイムに変更できます。設定は自動的に保存され、次回起動時にも維持されます。

| 設定項目 | 選択肢 | 説明 |
| :--- | :--- | :--- |
| **テーマ（背景色）** | 和紙 / 明 / 暗 / 漆黒 | 部屋の明るさに合わせて配色テーマを選択できます。デフォルトは目に優しい「和紙（セピア）」です。 |
| **書体（フォント）** | 明朝体 / ゴシック体 | 小説に適した美しい縦書き「明朝体（Noto Serif JP）」と、すっきりした「ゴシック体」を選択できます。 |
| **ページの送り方向** | 右から左（標準） / 左から右 | 縦書きテキストの送り方向を伝統的なRTL（右から左）または現代的なLTR（左から右）へ切り替えます。 |
| **文字サイズ** | 小 / 中 / 大 / 特大 | 本文テキストの文字サイズを4段階で調節します。 |
| **行間** | 狭い / 標準 / 広い | 行と行の間のスペースを3段階で調節します。 |
| **文字間** | 狭い / 標準 / 広い | 文字と文字の間のスペースを3段階で調節します。 |

## 5. しおり自動保存機能 (Auto Bookmark Persistence)

ゆうぞらには、読書位置を見失わないための**自動しおり保存機能**が備わっています。

- **自動保存タイミング**: ページをめくった時、プログレスバーを操作した時、ブラウザのタブを閉じた時やバックグラウンドに切り替えた時に、自動的に現在の読了進捗が保存されます。
- **セッションの自動復元**: アプリを再度開いた際、最後に読んでいた作品のファイル内容、進捗位置（ページ）、および設定（テーマや文字サイズ等）が自動的に読み込まれ、前回の続きからすぐに読書を再開できます。
