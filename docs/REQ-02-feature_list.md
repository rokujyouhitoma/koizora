# [REQ-02] 機能一覧 (Feature List) - ゆうぞら (Yuzora)

本ドキュメントは、青空文庫縦書きビューアー「ゆうぞら (Yuzora)」で実装されている、あるいは今後実装予定の全機能を一覧化し、そのステータス、仕様概要、および要求定義（URD）・要件定義（SRD）とのトレーサビリティ（追跡性）を管理する台帳です。
